==========================================================================

Die Benutzung von SeaMonkey ist an die Bedingungen der beiliegenden
Lizenzvereinbarung gebunden.

Diese ReadMe-Datei enthält Informationen über System-Voraussetzungen
sowie Installations-Anweisungen für SeaMonkey unter Microsoft Windows.

Mehr Informationen über SeaMonkey finden Sie unter
http://www.seamonkey-project.org/
Um Fehler und sonstiges Feedback zu melden, sollten Sie sich die Seite
http://www.seamonkey-project.org/dev/get-involved ansehn und sich mit 
Bugzilla unter https://bugzilla.mozilla.org/ vertraut machen, wo Sie
bereits gemeldete Fehler, Richtlinien zum Erstellen von Fehlerberichten
und mehr finden.

Um Hilfe mit Bugzilla zu bekommen, können Sie Sich mit Ihrem IRC-Client in
#mozillazine auf irc.mozilla.org melden.

====================================================

                 SeaMonkey erhalten

====================================================

Sie können Sich sogenannte "nightly builds" von SeaMonkey vom mozilla.org 
FTP-Server herunterladen.

  http://ftp.mozilla.org/pub/mozilla.org/seamonkey/nightly/

Die aktuellsten Builds finden Sie unter

  http://ftp.mozilla.org/pub/mozilla.org/seamonkey/nightly/latest-trunk/

Beachten Sie, dass es sich bei diesen "nightly builds" um Entwicklungsversionen 
handelt, welche von mozilla.org-Entwicklern zum Testen verwendet werden und 
somit Fehler enthalten können. Wenn Sie lieber eine stabile Version von 
SeaMonkey benutzen möchten, empfiehlt es sich die offiziellen 
Release-Versionen zu verwenden, welche vom SeaMonkey-Projekt in regelmäßigen
Abständen herausgegeben werden. Diese finden Sie unter

  http://www.seamonkey-project.org/releases/

Lesen Sie auch die Release-Notes für SeaMonkey in denen Sie Informationen zu 
bekannten Problemen und möglichen Problemen bei der Installation von SeaMonkey
nachlesen können. Die Release-Notes finden Sie unter der oben angeführten URL, 
unter der Sie auch die Release-Versionen finden.

====================================================

               Systemvoraussetzungen

====================================================

* Windows

   - Windows 2000, XP oder Vista
   - Intel Pentium Prozessor (500 MHz oder schneller)
   - 128 MB RAM
   - 50 MB freier Speicherplatz


====================================================

            Installationsanleitung

====================================================
Es wird empfohlen, alle Programme zu schließen bevor Sie das Setup starten.
Sie sollten weiterhin alle Antivirus-Software für die Dauer der Installation
deaktivieren.

Installieren Sie in ein sauberen (neues) Verzeichnis. Wenn Sie eine bestehende
Installation überschreiben kann dies zu Problemen führen.

Hinweis: Diese Anleitung erklärt nicht, wie Sie SeaMonkey selbst bauen können.
Informationen, wie Sie SeaMonkey aus dem mozilla.org - Sourcecode selbst 
erstellen können, finden Sie unter

  http://developer.mozilla.org/en/docs/Build_Documentation


Windows Installationanleitung
-----------------------------

Anmerkung: Für Windows 2000/XP Systeme, brauchen Sie
Administratorenrechte um SeaMonkey zu installieren. Sollten
Sie während der Installation einen "Error 5" erhalten,
stellen Sie sicher das die Installation mit
Administratorenrechten durchgeführt wird.

    Um SeaMonkey über den heruntergeladenen SeaMonkey-Installer
    zu installieren, folgen Sie folgenden Schritten:

	1. Laden Sie sich die Datei seamonkey-x.xx.en-US.win32.installer.exe
	   von der entsprechenden Seite auf Ihren Computer.

	2. Öffnen Sie den Ordner in welchem Sie die Datei abgespeichert 
	   haben und doppelklicken Sie die Datei, um mit der Installation 
           zu beginnen.

	3. Folgen Sie den Anweisungen des Setupprogramms am
     	   Bildschirm. SeaMonkey startet danach automatisch.


    Um Mozilla manuell duch die heruntergeladene
    .zip-Datei zu installieren, folgen Sie folgenden
    Schritten:

	1. Laden Sie sich die Datei seamonkey-x.xx.en-US.win32.zip
	von der entsprechenden Seite auf Ihren Computer.

	2. Öffnen Sie den Ordner in welchem Sie die Datei abgespeichert 
	haben und doppelklicken Sie die Datei, um diese zu öffnen.

	Anmerkung: Dieser Schritt geht davon aus, dass Sie
	bereits eine Version von WinZip installiert haben und
	wissen wie man damit umgeht. Falls nicht dann bekommen
	Sie WinZip und Informationen darüber unter
	www.winzip.de.

	3. Extrahieren Sie die .zip-Datei in ein Verzeichnis
	wie C:\Programme\mozilla.org\SeaMonkey.

	4. Um SeaMonkey zu starten, öffnen Sie den Ordner, in den
	Sie SeaMonkey extrahiert haben und doppelklicken die Datei
	SeaMonkey.exe.
