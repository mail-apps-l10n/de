Benutzerprofile verwalten

Wenn Sie das Internet geschäftlich und privat nutzen, möchten Sie wahrscheinlich unterschiedliche Lesezeichen, Voreinstellungen, Adressbücher und E-Mail-Einstellungen benutzen. Genauso können sich alle Mitglieder Ihrer Familie eine Kopie von Mozilla teilen, aber jeder kann seine eigenen Einstellungen für das Internet benutzen.

Der Profilmanager erlaubt es Ihnen verschiedene Profile zu erstellen. In einem Profil werden z.B. Lesezeichen, Voreinstellungen, E-Mail-Einstellungen und so weiter gespeichert. Beim allerersten Start von Mozilla auf Ihrem Computer wird automatisch ein Standardprofil angelegt. Wenn Sie zusätzliche Profile anlegen, werden Sie bei jedem Start von Mozilla gefragt, welches Profil Sie benutzen möchten.

In diesem Abschnitt:
  • Ein neues Benutzerprofil erstellen
  • Ein bestehendes Benutzerprofil löschen oder umbenennen
  • Externe Benutzerprofile (Roaming)
  • Roaming Einstellungen
  • Roaming Eintragsauswahl

Ein neues Benutzerprofil erstellen

So erstellen Sie ein neues Benutzerprofil:

  1. Öffnen Sie den Profilmanager:
    • Falls &brandShortName; gerade ausgeführt wird: Öffnen Sie das Menü Extras und klicken Sie auf Profil wechseln.
    • Falls &brandShortName; geschlossen ist: Klicken Sie im Startmenü auf Programme, dann auf &brandShortName;, und auf Profilmanager. Drücken Sie die Options-Taste während Sie &brandShortName; vom Finder oder Dock aus starten. Geben Sie folgendes an der Kommandozeile ein: ./mozilla -profilemanager
  2. Klicken Sie auf Profile verwalten und dann auf Profil erstellen. Lesen Sie sich die angezeigten Informationen durch und klicken Sie auf Weiter.
  3. Geben Sie einen Namen für das neue Profil ein. Sie können einen beliebigen Namen eingeben, z.B. Ihren richtigen Namen oder einen Namen, der beschreibt für was Sie das Profil benutzen wollen, z.B.Schule.
  4. Akzeptieren Sie das vorgeschlagene Verzeichnis zum Speichern Ihres Profils, oder wählen Sie ein anderes Verzeichnis mit Hilfe von Ordner auswählen aus.
  5. Wenn Sie Sprachpakete installiert haben, klicken Sie auf Sprachauswahl und wählen Ihre Sprache aus.
  6. Klicken Sie auf Fertigstellen.

Ein bestehendes Benutzerprofil löschen oder umbenennen

Sie möchten vielleicht Profile löschen, die sie normalerweise nicht benutzen. So löschen Sie ein bestehendes Profil oder benennen es um:

  1. Öffnen Sie den Profilmanager:
    • Falls &brandShortName; gerade ausgeführt wird: Öffnen Sie das Menü Extras und klicken Sie auf Profil wechseln.
    • Falls &brandShortName; geschlossen ist: Klicken Sie im Startmenü auf Programme, dann auf &brandShortName;, und auf Profilmanager. Drücken Sie die Options-Taste während Sie &brandShortName; vom Finder oder Dock aus starten. Geben Sie folgendes an der Kommandozeile ein: ./mozilla -profilemanager
  2. Klicken Sie auf Profile verwalten.
  3. Zum Löschen eines Profils, markieren Sie seinen Namen und klicken auf Profil löschen. Im darauffolgenden Bestätigungsdialog, wählen Sie eine der folgenden Optionen aus:
    • Dateien löschen: Löscht das komplette Profilverzeichnis inklusive aller gespeicherten Daten(Bookmarks, Einstellungen, E-Mails, usw.). Stellen Sie sicher, dass Sie in Zukunft keine Daten mehr von diesem Profil benötigen, bevor Sie diese Option auswählen.
    • Dateien nicht löschen: Entfernt das Profil nur aus der Liste der vorhandenen Profile, löscht aber nicht die Daten des Profils. Wenn Sie diese Option wählen werden die Daten des Profils nicht von der Festplatte gelöscht.
    Um ein Profil umzubenennen markieren Sie seinen Namen und klicken auf Profil umbenennen. Folgen sie anschließend den angezeigten Anweisungen.

Externe Benutzerprofile (Roaming)

Externe Benutzerprofile erlauben es Ihnen Ihr Profil und alle zugehörigen Einstellungen auf einem zentralem Server oder einer Datei-Freigabe zu speichern. Das ermöglicht es Ihnen, von mehreren Computern aus auf Ihr Profil zuzugreifen. Sie können damit zum Beispiel dieselben Lesezeichen, Adressbücher und andere Einstellungen sowohl Zuhause und auf der Arbeit nutzen. Weiters ermöglich dies Firmen, deren Mitarbeitern auf allen Computern immer Ihr persönliches &brandShortName; Profil zu nutzen.

Externe Benutzerprofile können auf einem HTTP oder einem FTP-Server gespeichert werden. Sie benötigen einen Benutzernamen und ein Passwort auf diesem Server, sodass Sie Schreibrechte in dem ausgewählten Verzeichnis haben, um Ihr Profil darin zu speichern oder es von dort aufzurufen. Alternativ können Sie in einem Netzwerk Ihr Profil auch auf einer Dateifreigabe speichern und mittels der Roaming-Einstellung Datei kopieren von dort laden.

Jedesmal wenn Sie &brandShortName; starten wird Ihr Profil heruntergeladen und sobald Sie &brandShortName; beenden wird Ihr Profil wieder am Server aktualisiert.

Roaming Einstellungen

Dieser Abschnitt erklärt die Verwendung der Roaming-Einstellungen. Sollten Sie den Einstellungs-Dialog noch nicht vor sich sehn, gehen Sie wie folgt vor:

  1. Öffnen Sie das Menü &brandShortName; Bearbeiten und wählen Sie Einstellungen.
  2. Klicken Sie auf die Kategorie Roaming-Benutzer.
  3. Aktivieren Sie die Einstellung Remote-Profilespeicherung aktivieren um Roaming für Ihr aktuelles Benutzerprofil zu nutzen.
  4. Wählen Sie eine der folgenden Möglichkeiten:
    • HTTP / FTP: Wählen Sie diese Einstellungen, wenn Sie Ihr Profil auf einem HTTP oder FTP-Server speichern wollen.
      • Basis-URL: Geben Sie hier die vollständige Adresse an, wo Ihr Profil gespeichert werden soll. Dies kann eine FTP, HTTP oder HTTPS Adresse sein. FTP ist im privaten Umfeld am einfachsten zu realisieren. In allen anderen Fällen, speziell bei der Übertagung über das Internet, empfiehlt es sich HTTPS zu nutzen, da es ansonsten möglich ist, dass Ihre gepeicherten Passwörter oder Lesezeichen (je nachdem welche Teile des Profils Sie übertragen) von anderen Personen gelesen werden.

        Beispiele:
        <https://user@http-server/dir/>
        <ftp://user@ftp-server//home/user/dir/>
        <ftp://user@ftp-server/dir/>

      • Benutzername: Geben Sie hier den Benutzernamen an, welchen Sie beim Zugriff auf den FTP oder HTTP Server benutzen möchten.
      • Passwort speichern: Aktivieren Sie diese Option, wenn Sie möchten, dass &brandShortName; sich das Passwort für den Roaming-Zugriff merkt.
      • Passwort: Geben Sie hier das Passwort für den oben angegebene Benuternamen ein.
    • Datei kopieren: Wählen Sie diese Option, wenn Sie ihr Profile auf einem lokalem Verzeichnis speichern möchten.
      • Klicken Sie auf Durchsuchen und wählen Sie das Verzeichnis, wo Sie Ihr Profile speichern wollen.

Roaming Eintragsauswahl

Standardmässig werden nur Lesezeichen, Cookies un das Persönliche Adressbuch am Roaming-Server gespeichert. Sollten alle Einträge ausgegraut sein, stellen Sie sicher, dass Sie Roaming aktiviert haben. Um das Speichern von Profile-Einstellungen via Roaming zu aktivieren oder zu deaktivieren, folgen Sie diesen Anweisungen:

  1. Öffnen Sie das Menü &brandShortName; Bearbeiten und wählen Sie Einstellungen.
  2. Unter der Kategorie Roaming-Benutzer wählen Sie die Unterkategorie Eintragsauswahl. (Sind keine Unterkategorien vorhanden, doppelklicken Sie auf Roaming-Benutzer um diese anzuzeigen.)
  3. Wählen Sie die Einträge, welche in Ihrem Roaming-Profile gespeichert werden sollen.

Es ist nicht möglich Ihre lokalen Emails mittels Roaming abzugleichen, dies würde sehr lange Abgleichzeiten bedeuten. Benutzen Sie für diese Fälle einen IMAP-Server.

Hinweis: Wenn Sie eine langsame Netzwerk-Verbindung besitzen, kann der Start von &brandShortName; sehr lange dauern, da beim Start alle Daten Ihres Roaming-Profiles von Ihrem Roaming-Server heruntergeladen werden müssen und einige Dateien sehr gross sein können. &brandShortName; muss alle Ihre Datein, welche Sie ausgewählt haben herunterladen um voll funktionsfähig zu sein. Beim Beenden von &brandShortName; werden Ihre Roaming-Dateien wieder auf den Server gespeichert.

Für mehr Informationen bezüglich Einrichtung von Roaming-Zugriffen auf einem Server, siehe mod_roaming - Roaming Access module for Apache(englisch) oder Manually Implementing Roaming Access(englisch).
